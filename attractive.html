A Lightweight Library for Declarative DOM Actions

Attractive.js is a tiny JavaScript library that lets you add interactivity to your web pages using simple HTML attributes. No JavaScript knowledge required! This page demonstrates ALL features with interactive examples.

üîñ Attributes

Add, remove, toggle, and cycle through HTML attributes on elements.

addAttribute

Add an attribute to an element. This example adds the 'disabled' attribute to an input field.

<button data-action="addAttribute" data-target="#input1" data-attribute="disabled"> Disable Input </button>

removeAttribute

Remove an attribute from an element. Re-enable the input field above.

<button data-action="removeAttribute" data-target="#input1" data-attribute="disabled"> Enable Input </button>

toggleAttribute

Toggle an attribute on/off. Works great with the 'disabled' attribute or the 'open' attribute on details elements.

Click to expand details

This is the hidden content inside the details element!

<button data-action="toggleAttribute" data-target="#input2" data-attribute="disabled"> Toggle Disabled </button> <button data-action="toggleAttribute" data-target="#details1" data-attribute="open"> Toggle Details </button>

cycleAttribute

Cycle through different attribute values. This example toggles a password input between visible and hidden.

<button data-action="cycleAttribute" data-target="#passwordInput" data-attribute="type" data-values="password text"> Toggle Password Visibility </button>

üé® Classes

Add, remove, toggle, and cycle through CSS classes for styling elements.

addClass

Add a CSS class to an element to change its appearance.

I'm a box! Click the button to add background color.
<button data-action="addClass" data-target="#box1" data-class="bg-primary"> Add Background </button>

removeClass

Remove a CSS class from an element.

<button data-action="removeClass" data-target="#box1" data-class="bg-primary"> Remove Background </button>

toggleClass (Single)

Toggle a single class on/off.

Toggle my background color!
<button data-action="toggleClass" data-target="#box2" data-class="bg-primary"> Toggle Background </button>

toggleClass (Multiple)

Toggle multiple classes at once by separating them with spaces.

Toggle multiple classes on me!
<button data-action="toggleClass" data-target="#box3" data-class="bg-black text-white"> Toggle Dark Theme </button>

cycleClass

Cycle through multiple CSS classes. Each click applies the next class in the sequence.

I cycle through colors!
<button data-action="cycleClass" data-target="#box4" data-class="bg-black bg-red-500 bg-blue-500"> Cycle Colors </button>

üìã Clipboard

Copy text to clipboard with visual feedback.

copy (Direct Value)

Copy a string directly using the data-value attribute.

Click the button and paste somewhere to see the copied URL!

<button data-action="copy" data-value="https://attractivejs.railsdesigner.com"> Copy Website URL </button>

copy (From Target)

Copy the value from an input field using data-target.

<button data-action="copy" data-target="#copyInput"> Copy Input Value </button>

copy (With Success Feedback)

Visual feedback when copy succeeds. The element gets a 'copy-success' class and data-copy-success attribute.

<button data-action="copy" data-target="#copyInput2" data-copy-duration="2000"> Copy with Feedback </button> /* CSS for feedback */ [data-copy-success="true"] { border-color: var(--color-success); color: var(--color-success); }

copy (Text with Spaces)

Use backticks to copy text containing spaces.

<button data-action="copy" data-value="`Hello World! This is attractive.js`"> Copy Text with Spaces </button>

üí¨ Dialog

Open and close HTML dialog elements, including modal dialogs with backdrops.

dialog#open (Regular Dialog)

Open a non-modal dialog element that can be dismissed by clicking outside.

Regular Dialog

This is a regular (non-modal) dialog. You can click outside to close it, or use the button below.
<button data-action="dialog#open" data-target="#myDialog"> Open Regular Dialog </button> <dialog id="myDialog"> <button data-action="dialog#close" data-target="#myDialog">Close</button> </dialog>

dialog#openModal (Modal Dialog)

Open a modal dialog with a backdrop. Can only be closed via buttons, not by clicking outside.

Modal Dialog

This is a modal dialog with a backdrop. You must use a button to close it.
<button data-action="dialog#openModal" data-target="#modalDialog"> Open Modal Dialog </button> <dialog id="modalDialog"> <form method="dialog"> <!-- Form content --> </form> </dialog>

üìù Form

Reset and submit forms programmatically, even from outside the form element.

form#reset

Reset a form to its initial values. The reset button can be outside the form!

<button data-action="form#reset" data-target="#demoForm"> Reset Form </button>

form#submit (Auto-submit on Change)

Automatically submit a form when a select element changes. Great for filter forms!

Change the select to auto-submit the form!

<select data-action="change->form#submit" data-target="#filterForm"> <option>All Categories</option> </select>

form#submit (With Delay/Debounce)

Debounce form submission using data-submit-delay. Useful for search-as-you-type.

Form submits 1 second after you stop typing!

<input data-action="input->form#submit" data-target="#searchForm" data-submit-delay="1000">

üéØ Events & Custom Triggers

By default, attractive.js uses sensible event triggers for each element type:

button, a ‚Üí click event
form ‚Üí submit event
select ‚Üí change event
input ‚Üí input event

Override Default Events

You can specify custom events using the arrow notation:

<button data-action="mouseover->addClass" data-class="highlight">Hover me</button>
<input data-action="focus->addAttribute" data-attribute="data-focused" data-value="true">
<div data-action="mouseenter->toggleClass" data-class="active">Hover zone</div>

Use any valid DOM event: click, mouseover, mouseenter, mouseleave, focus, blur, input, change, submit, etc.

üéØ Targets & Selectors

The data-target attribute accepts any CSS selector to specify which element(s) to affect:

Selector Examples

data-target="#myElement" ‚Üí Target by ID
data-target=".my-class" ‚Üí Target all elements with class
data-target="button" ‚Üí Target all button elements
data-target=".container .item" ‚Üí Nested selector
data-target="[data-type='special']" ‚Üí Attribute selector

Self-Targeting

If you don't provide a data-target, the action applies to the element itself:

<button data-action="toggleClass" data-class="active"> This button toggles the 'active' class on itself </button>

üì¶ Installation & Setup

CDN (Easiest)

Add this script tag to your HTML:

<script src="https://cdn.jsdelivr.net/npm/attractivejs" defer></script>

NPM

Install via npm:

npm install attractivejs

Then import in your JavaScript:

import 'attractivejs';

That's It!

Once the library is loaded, all data-action attributes will automatically work. No initialization code needed!
